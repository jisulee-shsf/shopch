## Shopch 🦈
- 마치 기술(tech)을 쇼핑(shop)하듯, 다양한 기술을 활용한 주문 기능 중심의 개인 프로젝트입니다.
- 테스트 코드 작성과 성능 최적화를 목표로 기획부터 설계, 개발, 문서화, 배포, 리팩토링까지 전 과정을 수행했습니다.
- 추후 협업을 대비해 이슈와 PR을 생성하여 체계적으로 관리했습니다.
